# Best Xcode Theme
Yet another best theme for Xcode

## Installation

1. Clone the repository

```bash
git clone git@github.com:patrykwas/best-xcode-theme.git
```

2. Run the install script inside the cloned repository

```bash
sh Install.sh
```
